Read below for maximums. If you need to raise these limits, please contact us.

## Per-account limits

### Queue
- 5000 enqueued requests per second
- 250 concurrent queue consumers
- 100 messages per batch
- 100 retries per message

### Database

- 10 GB database size

### Environment variables

- 128 environment variable key/value pairs
- 5 KB per environment variable key/value pair

## Per-crawler limits

### Table

- Unlimited rows per table
- 100 columns per table
- 2 MB per row

### Bundle

- 10MB minified bundle size
